What's inside

Two sides of the same key

DriveOn pairs drivers looking for a ride with owners renting out their cars — each with a flow built end-to-end.

Driver flow

Browse and map nearby cars, filter, book a slot, pay on a ticket-stub checkout, then unlock with an NFC digital key — with mandatory fuel-check verification at pickup and drop-off.

  • Search & map
  • Booking
  • Digital key
  • Fuel verification
  • Transit navigation

Owner flow

An earnings dashboard with animated revenue, approve / deny requests, a redesigned week-view schedule, rental history, a garage of your cars, and a list-a-car wizard.

  • Earnings
  • Approve / deny
  • Week schedule
  • Garage
  • Listings

Design system

An exact turquoise → deep-teal token palette, system-font uppercase display type, Lucide iconography, and a 35-component Figma library — light/dark with four accent themes.

  • HSL tokens
  • Lucide icons
  • Light / Dark
  • Component library

Figma → vibe-coded

Designed as a component library in Figma — now a fully wired, clickable prototype where every element reacts and both flows play start-to-finish — then vibe-coded into this live React prototype with Claude, kept in lockstep so every screen, token and component matches its Figma source 1:1.

  • Figma library
  • Interactive prototype
  • Vibe-coded with Claude
  • 1:1 in sync
